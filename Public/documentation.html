API Documentation

This documentation outlines the available endpoints for the movie API and provides details about how to use each one.

Method Endpoint Description Details
GET /movies Return a list of ALL movies Response: 
[
  {
    "_id": "641e8bf5d22e42419cabc123",
    "Title": "Inception",
    "Description": "A thief who steals corporate secrets through the use of dream-sharing technology...",
    "Genre": { "Name": "Sci-Fi", "Description": "Science fiction films" },
    "Director": { "Name": "Christopher Nolan", "Bio": "British-American film director..." },
    "Actors": ["Leonardo DiCaprio", "Joseph Gordon-Levitt"],
    "ImagePath": "/images/inception.jpg",
    "Featured": true
  },
  ...
]
GET /movies/:title Return data about a single movie by title Path Parameter: title (string)
Response: 
{
  "_id": "641e8bf5d22e42419cabc123",
  "Title": "Inception",
  "Description": "A thief who steals corporate secrets through the use of dream-sharing technology...",
  "Genre": { "Name": "Sci-Fi", "Description": "Science fiction films" },
  "Director": { "Name": "Christopher Nolan", "Bio": "British-American film director..." },
  "Actors": ["Leonardo DiCaprio", "Joseph Gordon-Levitt"],
  "ImagePath": "/images/inception.jpg",
  "Featured": true
}
GET /genres/:name Return data about a genre by name Path Parameter: name (string)
Response: 
{
  "Name": "Sci-Fi",
  "Description": "Science fiction films"
}
GET /directors/:name Return data about a director by name Path Parameter: name (string)
Response: 
{
  "Name": "Christopher Nolan",
  "Bio": "British-American film director, producer, and screenwriter...",
  "BirthYear": 1970
}
POST /users Register a new user Request Body: 
{
  "Username": "johndoe",
  "Password": "password123",
  "Email": "johndoe@example.com",
  "Birthday": "1990-01-01"
}
Response: 
{
  "_id": "641e8bf5d22e42419cabc678",
  "Username": "johndoe",
  "Email": "johndoe@example.com",
  "Birthday": "1990-01-01T00:00:00.000Z",
  "FavoriteMovies": []
}
PUT /users/:username Update a user's info Path Parameter: username (string)
Request Body: 
{
  "Username": "johnupdated",
  "Password": "newpassword",
  "Email": "johnupdated@example.com",
  "Birthday": "1991-01-01"
}
Response: 
{
  "_id": "641e8bf5d22e42419cabc678",
  "Username": "johnupdated",
  "Email": "johnupdated@example.com",
  "Birthday": "1991-01-01T00:00:00.000Z",
  "FavoriteMovies": []
}
POST /users/:username/movies/:movieId Add a movie to a user's favorites Path Parameters: username (string), movieId (string)
Response: 
{
  "_id": "641e8bf5d22e42419cabc678",
  "Username": "johndoe",
  "FavoriteMovies": ["641e8bf5d22e42419cabc123"]
}
DELETE /users/:username/movies/:movieId Remove a movie from a user's favorites Path Parameters: username (string), movieId (string)
Response: 
{
  "_id": "641e8bf5d22e42419cabc678",
  "Username": "johndoe",
  "FavoriteMovies": []
}
DELETE /users/:username Deregister a user Path Parameter: username (string)
Response: 
"johndoe was deleted."

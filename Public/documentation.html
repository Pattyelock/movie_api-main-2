Welcome to My Movie API Documentation

Here you will find information about the API and its endpoints.

API Endpoints

Request URL Endpoint Description Response
GET /movies Retrieve a list of all movies Array of movie objects
GET /movies/:title Retrieve details of a movie by title Movie object
GET /genres/:name Retrieve genre details by name Genre object
GET /directors/:name Retrieve director details by name Director object
POST /users Register a new user Confirmation of registration
PUT /users/:username Update user information Confirmation of update
POST /users/:username/favorites/:movieId Add a movie to user’s favorites Confirmation of addition
DELETE /users/:username/favorites/:movieId Remove a movie from user’s favorites Confirmation of removal
DELETE /users/:username Deregister user Confirmation of account deletion

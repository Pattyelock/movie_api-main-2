Welcome to My Movie API Documentation

Here you will find information about the API and its endpoints.

Base URL: `http://localhost:3000` (or your live API URL)

API Endpoints

Request URL Endpoint Description Response
GET /movies Retrieve a list of all movies 
[
  {
    "Title": "Inception",
    "Description": "A mind-bending thriller where dream infiltration is a reality.",
    "Genre": { "Name": "Sci-Fi", "Description": "Science fiction" },
    "Director": { "Name": "Christopher Nolan", "Bio": "English film director, producer, screenwriter." },
    "Actors": ["Leonardo DiCaprio", "Joseph Gordon-Levitt", "Ellen Page"],
    "ImagePath": "/images/inception.jpg",
    "Featured": true
  },
  ...
]
GET /movies/:title Retrieve details of a movie by title (e.g., /movies/Inception) 
{
  "Title": "Inception",
  "Description": "A mind-bending thriller where dream infiltration is a reality.",
  "Genre": { "Name": "Sci-Fi", "Description": "Science fiction" },
  "Director": { "Name": "Christopher Nolan", "Bio": "English film director, producer, screenwriter." },
  "Actors": ["Leonardo DiCaprio", "Joseph Gordon-Levitt", "Ellen Page"],
  "ImagePath": "/images/inception.jpg",
  "Featured": true
}
GET /genres/:name Retrieve genre details by name (e.g., /genres/Sci-Fi) 
{
  "genre": "Sci-Fi",
  "movies": [
    {
      "Title": "Inception",
      "Description": "A mind-bending thriller where dream infiltration is a reality.",
      "Director": "Christopher Nolan",
      "Actors": ["Leonardo DiCaprio", "Joseph Gordon-Levitt", "Ellen Page"],
      "ImagePath": "/images/inception.jpg"
    }
  ]
}
GET /directors/:name Retrieve director details by name (e.g., /directors/Christopher Nolan) 
{
  "director": "Christopher Nolan",
  "bio": "English film director, producer, screenwriter.",
  "birthYear": 1970,
  "deathYear": null
}
POST /users Register a new user 
{
  "message": "User newUser123 registered successfully."
}
PUT /users/:username Update user information 
{
  "message": "User newUser123 information updated."
}
POST /users/:username/favorites/:movieId Add a movie to user’s favorites 
{
  "message": "Inception has been added to newUser123's favorites."
}
DELETE /users/:username/favorites/:movieId Remove a movie from user’s favorites 
{
  "message": "Inception has been removed from newUser123's favorites."
}
DELETE /users/:username Deregister user 
{
  "message": "User newUser123 deregistered."
}
